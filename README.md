# Freddy - rebel machines slayer

A project for the GamedevJs jam 2026, the theme is "Machines".

## 🚀 Quick setup

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```

## 📦 Script

- `npm run dev` - Start development server
- `npm run build` - Production build
